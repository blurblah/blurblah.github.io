Wordpress에서 글을 쓸 때

깔끔하게 코드 삽입할 수 있는 기능이 없어서 플러그인을 좀 찾아봤다.

Syntaxhighlighter 라는게 있길래 일단은 설치.

아래는 테스트.

```
[java]
public void foo() {
...
}
[/java]
```

스크롤이 신경쓰이는데 수정해서 쓰던가 해야겠다.
